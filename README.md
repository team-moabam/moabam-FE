## 개발 서버 실행 방법

**패키지 설치**

```sh
npm install
```

**개발 서버 실행**  

로컬 환경에서 Mock API를 실행하기 위해서는 아래 커맨드로 msw를 활성화할 수 있습니다.  

```sh
npm run dev:msw # msw를 활성화하여 개발 서버 실행
```
